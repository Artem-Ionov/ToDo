Чтобы запустить этот проект локально на компьютере, выполните следующее:
1. Настройте среду разработки python (версия указана в файле runtime.txt) и MySQL 8.0.41.0.
2. Рекомендуется настроить виртуальное окружение. Для этого в терминале выполните следующие команды:
     python -m venv название_окружения
     название_окружения\Scripts\activate
3. Установите необходимые зависимости (указаны в файле requirments.txt). Для этого можно выполнить в терминале команду: 
  pip install -r requirements.txt.
4. Создайте для проекта базу данных в MySQL.
5. В корне проекта создайте файл .env со следующим содержимым:
     NAME=имя вашей базы данных
     USER=имя пользователя MySQL
     PASSWORD=ваш пароль
     HOST=localhost
6. Выполните в терминале следующие команды:
     python manage.py migrate
     python manage.py runserver
